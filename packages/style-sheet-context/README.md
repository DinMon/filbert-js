# @filbert-js/style-sheet-context

> StyleSheet Context for filbert-js

## Install

```bash
yarn add @filbert-js/style-sheet-context
```

## Usage

```jsx
import {
  StyleSheetProvider,
  StyleSheetContext,
} from '@filbert-js/style-sheet-context';
```

More documentation is available at https://filbert-js.vercel.app.
